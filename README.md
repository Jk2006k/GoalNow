# Intbar

A full-stack interview system designed to simulate real-time technical interviews using AI, code execution APIs, and proctoring features. The platform evaluates candidates through coding, voice interaction, and behavioral monitoring.

---

## Overview

Intbar is built to replicate a real interview environment where users can attempt coding questions, receive AI-driven interactions, and be monitored for fairness. It integrates external APIs and real-time analysis to provide a comprehensive evaluation system tailored for organizations.

---

## Tech Stack

* **Frontend:** React
* **Backend:** Node.js with Express
* **Database:** MongoDB
* **Code Execution:** Judge0 API
* **AI Integration:** Google Gemini API, Groq API
* **Additional:** WebRTC / Media APIs for camera and voice processing

---

## Key Features

* Real-time coding interview environment with multi-language support
* AI-powered interview interaction using Gemini and Groq APIs
* Live camera monitoring and proctoring with video recording
* Voice recognition for verbal responses
* Timed questions based on difficulty levels
* Behavioral monitoring for fairness and integrity
* Secure and scalable architecture
* Real-time code execution and feedback

---

## How the Platform Works

1. **User Interaction** – Candidates access the platform and participate in interview sessions with dynamically displayed questions
2. **Code Execution** – Real-time code compilation and execution with instant feedback
3. **AI Evaluation** – Intelligent assessment using Gemini and Groq APIs for comprehensive evaluation
4. **Proctoring** – Real-time monitoring with video/screenshot capture and suspicious activity detection
5. **Database Management** – Secure storage of user data, interview records, and results
